# ConsoleApp44 / MusicPlayerWpf

## Notes
- `MusicPlayerWpf` is a WPF desktop project and runs on Windows.
- Target framework: `net9.0-windows`.
- Build and run this project on Windows with .NET SDK installed.

## Typical commands (Windows)
- `dotnet build ConsoleApp44.sln`
- `dotnet run --project MusicPlayerWpf`
